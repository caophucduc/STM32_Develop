# STM32_Develop bootloader 2 apps.
Develop for STM32
STM32 bootloader example that can switch to 2 apps.

# Hardware
Using STM32F103C8T6
With sate of pin B11 is High(1) running app 2 and state is Low(0) will running with app 1
